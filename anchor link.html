heading tags – these range from h1 to h6, where heading h1 is largest in size, getting smaller as they move up h6
paragraph tags – are all enclosed by using the paragraph tag.
list tags – have different variations. Use the order list tag for an ordered list, and use under list for an unordered list. Then, enclose individual list items using the list tag. The img tag is used to embed an image in an HTML page.
image tag are not technically inserted into a web page; images are linked to web pages. The img tag creates a holding space for the referenced image.
The img tag has two required attributes:
src - Specifies the path to the image
alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed
